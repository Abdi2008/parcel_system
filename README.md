# Instant Parcel Management System (IPMS)

A web-based logistics system built with PHP and MySQL. It allows users to book parcel deliveries, calculate dynamic prices based on service tiers (Standard/Deluxe), and track order status.

## Features
* **User Authentication:** Secure Login & Registration.
* **Booking Engine:** Dynamic pricing calculation (JavaScript + PHP).
* **Dashboard:** Separate views for Customers and Admins.
* **Order History:** Users can view their past bookings and status.

## How to Run
1.  Download the repository.
2.  Copy the folder to `xampp/htdocs`.
3.  Import `parcel_db.sql` into phpMyAdmin.
4.  Configure `db_connect.php` if necessary.
5.  Run `localhost/parcel_system`.

## Tech Stack
* HTML5, CSS3, JavaScript
* PHP (Backend)
* MySQL (Database)
